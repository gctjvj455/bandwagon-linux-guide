## 五、机房怎么选：线路差异比配置差异更影响体验

很多人买完才发现"怎么晚上卡成 PPT"，问题十有八九出在机房和线路上。搬瓦工的机房按线路大致可以这样理解：

- **CN2 GIA（AS4809）**：电信最贵最稳的线路，晚高峰也不容易丢包。CN2 GIA-E 系列默认走这条，国内用户体验最好。
- **CN2 GT（AS4809 Global Transit）**：比 CN2 GIA 便宜一档，2019 年后拥堵明显，白天还行，晚上体验一般。CN2 系列走这条。
- **CMIN2（China Mobile AS58807）+ China Unicom Premium（AS10099）**：移动、联通的高端线路，DC9 机房三网都走高端，是搬瓦工目前综合体验最好的机房之一。

## 七、买完之后：从零开始的 Linux 上手清单

机器到手了，接下来才是真正学 Linux 的部分。下面这份清单是我自己走过的路，按顺序做下来基本就能从"小白"过渡到"能自己折腾"。

**第一步：连上去。** Windows 用自带的 PowerShell 跑 `ssh root@你的IP`，Mac/Linux 直接终端敲就行。第一次连会问 yes/no，输 yes。密码在 KiwiVM 面板的 Root Password Modification 里重置一个你记得住的。

**第二步：换源 + 更新系统。** 国内访问默认源慢，换成国内镜像源（清华、阿里、中科大都行），然后 `apt update && apt upgrade -y`（Debian/Ubuntu）或 `dnf update -y`（CentOS 系）。这一步顺便就学了"包管理器"是什么。

**第三步：建一个普通用户，别老用 root。** `adduser 你的名字`，给它 sudo 权限，以后用普通用户登录，需要权限时加 sudo。这是 Linux 安全的第一课。

**第四步：改 SSH 端口、禁用密码登录、启用密钥登录。** 改 `/etc/ssh/sshd_config`，把 Port 22 改成一个高位端口，PasswordAuthentication 改成 no，重启 sshd。这一步练完，你基本就把"配置文件怎么改、服务怎么重启、防火墙怎么放行端口"全学了一遍。

**第五步：装一个 fail2ban 或 ufw，配防火墙。** 学安全的基础设施。

**第六步：开始装你感兴趣的东西。** Nginx、Docker、Node.js、Python 环境、MySQL……想学什么装什么，装坏了 KiwiVM 一键重装，三分钟满血复活。

学到这一步，你已经从"买哪台"过渡到"能用 Linux 做点什么"了——这才是买 VPS 的真正意义。
